Store Managment System Project Built as a C++ Console Application,\
on visual studio (solution provided) can be compiled and run on non-VS IDE.

The program acts like an online store system (such as ebay,amazon, etc)\
it allows to add users as buyers/sellers and buy/sell items and give feedback.\
it also stores data in the database in the form of a txt file that comes included.\
upon closing the program all added data will be saved into the txt file.

Console Instructions:\ 
no special insturction needed, choose an option from the menu.
